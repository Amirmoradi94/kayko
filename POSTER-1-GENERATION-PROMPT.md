## Main Prompt (DALL-E 3 / ChatGPT Image Generation)

```
Create a professional, realistic browser screenshot-style poster for a Chrome extension called "Kayko". 

SCENE COMPOSITION:
- Show the top portion of a modern web browser window (Chrome/Edge style)
- Browser address bar visible at top showing "chat.openai.com" or "claude.ai"
- Clean, minimalist browser interface with white/light gray background
- Professional, high-quality screenshot aesthetic

MAIN ELEMENTS:
1. Left side (60% of image):
   - Visible portion of ChatGPT or Claude AI chat interface
   - Show a textarea/chat input box with some typed text visible (like "Write a Python function to..." or similar prompt)
   - Next to the textarea, show a small, cute animated cat icon (orange/yellow cat face, #FFB84D color, about 64x64 pixels)
   - Cat icon has two small toggle buttons on its ears (left ear: green toggle ON, right ear: purple sparkle button)
   - Cat icon positioned on the right side of the textarea, floating slightly
   - Show subtle animation effect: cat icon has a green checkmark glow indicating "saved" state
   - Modern, clean AI chat interface with conversation history visible above

2. Right side (40% of image):
   - Large, bold headline text: "Never Lose Your Prompts Again" 
   - Font: Modern sans-serif (Inter, SF Pro, or Roboto), bold, 48-56px, color #1F2937 (dark gray)
   - Subheadline below: "Auto-save your AI prompts locally. Complete privacy. Zero configuration."
   - Font: Regular weight, 24-28px, color #6B7280 (medium gray)
   - Small feature bullets or icons showing: Auto-save, Privacy, Works Everywhere

3. Bottom section:
   - Platform logos/icons in a row: ChatGPT, Claude, Gemini, Grok, Perplexity (small, 40x40px each, semi-transparent)
   - Text: "Works on all major AI platforms"
   - Small "Install Free" button in blue (#3B82F6), rounded corners

COLOR SCHEME:
- Primary background: Clean white (#FFFFFF) or very light gray (#F9FAFB)
- Browser chrome: Light gray (#F3F4F6)
- Accent colors: Blue (#3B82F6), Green (#10B981), Orange (#FFB84D for cat)
- Text: Dark gray (#1F2937) for headlines, medium gray (#6B7280) for body

STYLE:
- Realistic browser screenshot aesthetic
- Professional, modern design
- High resolution, crisp details
- Subtle shadows and depth
- Clean typography
- Not cartoonish - realistic and professional
- Soft lighting, no harsh shadows
- Modern UI design (2024 style)

TECHNICAL:
- Aspect ratio: 2.5:1 (1400x560 pixels)
- High quality, sharp focus
- Professional product screenshot style
- Clean composition with proper spacing
```

## Post-Processing Tips

After generating the base image:

1. **Add Text Overlay**
   - Use design software (Figma, Photoshop, Canva)
   - Add headline and subheadline with exact fonts
   - Ensure high contrast and readability

2. **Add Platform Logos**
   - Download official platform logos
   - Resize to 40x40px
   - Arrange horizontally at bottom
   - Apply 70% opacity

3. **Add Kayko Cat Icon**
   - Use the actual Kayko icon from the extension
   - Position next to textarea
   - Add glow effect for "saved" state
   - Ensure proper scaling

4. **Color Correction**
   - Ensure consistent color scheme
   - Match brand colors (#3B82F6 blue, #10B981 green, #FFB84D orange)
   - Professional color grading

5. **Final Polish**
   - Sharpen text and icons
   - Ensure 1400x560px final size
   - Optimize file size (under 1MB)
   - Export as PNG for best quality
