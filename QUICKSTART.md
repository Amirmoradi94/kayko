## Common Tasks

### Recover a Lost Prompt
1. Click Kayko icon (next to textarea or in toolbar)
2. Find prompt in history
3. Click "Copy"
4. Paste back into chat

### Export All Prompts (Backup)
1. Click Kayko toolbar icon
2. Click "View Saved Prompts"
3. Click "Export" at bottom
4. Save JSON file

### Import Old Prompts
1. Open history panel
2. Click "Import"
3. Select previously exported JSON file
4. Prompts merged automatically

### Clear History
1. Open history panel
2. Click "Clear All" at bottom
3. Confirm (warning: cannot undo!)

### Exclude a Website
1. Open history panel
2. Click settings icon (âš™ï¸)
3. Add domain to "Excluded sites"
4. Example: `banking-site.com, private-chat.com`

## Troubleshooting (1-Minute Fixes)

**Icon not appearing?**
- Refresh page (F5)

**Prompt not saving?**
- Wait 3 seconds after typing
- Check if auto-save is enabled (Settings)

**Side panel won't open?**
- Click toolbar icon instead
- Reload extension at chrome://extensions/
