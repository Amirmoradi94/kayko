Privacy Policy for Kayko

Last Updated: November 2024

Introduction

Kayko ("we," "our," or "the extension") is a browser extension that automatically saves your LLM (Large Language Model) prompts locally in your browser. This Privacy Policy explains how we collect, use, and protect your information.

Key Principle: All data processing happens locally on your device. We do not operate any servers that collect or store your data.

Information We Collect

1. Prompt Text

We collect and store the text you type in LLM chat textareas, including:

  • The full text of your prompts
  • Metadata associated with each prompt (timestamp, platform name, page URL)
  • Favorite/bookmark status you assign to prompts

Note: This text may contain personally identifiable information (PII) if you include such information in your prompts.

2. User Activity

We monitor the following activities to provide auto-save functionality:

  • Typing events in textarea elements on LLM platforms
  • Clicks on the Kayko save icon
  • Interactions with the extension's side panel

3. Website Content

We access the following website content:

  • Text content from textarea elements (to save your prompts)
  • Page URL and hostname (to identify which LLM platform you're using)

4. Settings and Preferences

We store your extension settings locally, including:

  • Maximum number of prompts to keep
  • Auto-save enabled/disabled preference
  • OpenAI API key (if you choose to use the "Enhance Prompt" feature)

How We Use Your Information

  • Auto-save functionality: To automatically save your prompts as you type
  • History management: To display, search, and organize your saved prompts in the side panel
  • Platform identification: To correctly label and categorize prompts by LLM platform
  • Prompt enhancement: If you use the "Enhance Prompt" feature, your prompt text is sent to OpenAI's API to generate an improved version (see Third-Party Services below)

Data Storage

All data is stored locally in your browser using Chrome's local storage API. Your prompts never leave your computer unless you explicitly export them or use the "Enhance Prompt" feature.
  • Storage location: Browser's local storage (chrome.storage.local)
  • Data retention: Data persists until you delete it or uninstall the extension
  • Storage limits: Browser local storage has a size limit (~10MB), which typically allows for hundreds to thousands of prompts
  • Data export: You can export your prompts as JSON files at any time

Third-Party Services

OpenAI API (Optional Feature)

The "Enhance Prompt" feature is optional and requires you to provide your own OpenAI API key. When you use this feature:

  • Your prompt text is sent directly to OpenAI's API (https://api.openai.com)
  • We do not store or log your API key or API responses
  • OpenAI's privacy policy applies to data sent to their service
  • This feature is completely optional and disabled by default

OpenAI Privacy Policy: https://openai.com/policies/privacy-policy

Data Sharing

We do not share, sell, or transmit your data to any third parties, except:

  • When you explicitly use the "Enhance Prompt" feature (sends data to OpenAI API)
  • When you export your prompts (creates a local file on your device)

We do not:

  • Collect analytics or tracking data
  • Send data to our own servers (we don't operate any servers)
  • Share data with advertisers or data brokers
  • Use third-party analytics services

Data Security

We implement the following security measures:

  • All data is stored locally in your browser's secure storage
  • No network transmission of your prompts (except when using optional "Enhance Prompt" feature)
  • No external servers or databases
  • Your OpenAI API key (if provided) is stored locally and encrypted by Chrome's storage system

Note: While we take security seriously, no method of electronic storage is 100% secure. Your browser's local storage is protected by your operating system's security measures.

Your Rights and Choices

You have full control over your data:

  • View your data: Access all saved prompts through the extension's side panel
  • Delete individual prompts: Delete any prompt at any time
  • Clear all data: Delete all saved prompts using the "Clear All" button
  • Export your data: Export all prompts as a JSON file
  • Disable auto-save: Turn off automatic saving in settings
  • Uninstall: Uninstalling the extension removes all stored data

Children's Privacy

Kayko is not intended for children under 13 years of age. We do not knowingly collect personal information from children. If you are a parent or guardian and believe your child has provided us with personal information, please contact us so we can delete such information.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date. You are advised to review this Privacy Policy periodically for any changes.
